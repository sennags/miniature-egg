Sobre o Projeto

Todo dia um ovo. Todo dia uma oportunidade de aprender algo novo.

A Origem

O Miniature Egg é um projeto pessoal de estudos. Nada mais é do que minha prática diária de código — um jeito de aprender e evoluir com HTML, CSS e JavaScript.

A ideia de fazer um site de receitas de ovos nasceu como uma forma de praticar funcionalidades reais: temporizadores, temas, animações, responsividade. Coisas que eu realmente usaria no dia a dia.

O Que Fazemos

Hoje, o site reúne receitas testadas, temporizadores inteligentes e dicas que você só encontra em livros de culinária.

Cada receita foi pensada para ser simples, direta e com tudo que você precisa para acertar na primeira vez.

O Objetivo

Este é um projeto em constante evolução. O objetivo é crescer — mais receitas, mais funcionalidades, mais aprendizado.

📚 Tecnologias

  • HTML5 — Estrutura semântica
  • CSS3 — Estilos, animações e temas
  • JavaScript — Interatividade e temporizadores
  • Google Fonts — Tipografia personalizada
  • Web Audio API — Sons do alarme
